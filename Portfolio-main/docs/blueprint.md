# **App Name**: MD SUBBIR BIN HARUN Portfolio

## Core Features:

- Hero Section: Display a centered hero section with a profile image, introductory text, and call-to-action buttons.
- Skills Showcase: Showcase technologies and skills with animated logos and skill bars.
- Experience Timeline: Display a timeline of professional experiences with company logos and role descriptions using fade-in animations.
- Projects Portfolio: Showcase selected projects with hover tilt animations and tech badges, potentially sourcing data from a JSON file.
- About Section: Include a biography, photo, and animated skills summary.
- Contact Form: Implement a simple animated contact form with links to LinkedIn, GitHub, and email.
- AI-Powered Chatbot: Develop an AI chatbot to answer questions about your work using the provided CV as a tool.

## Style Guidelines:

- Primary color: Subtle purple (#A092FF) for a modern and professional aesthetic.
- Background color: Light gray (#F5F5F5), providing a clean and neutral canvas, with a subtle gradient effect as seen in the Brooklyn Gilbert example.
- Accent color: A contrasting shade of orange (#FFB347) for interactive elements, drawing focus to calls-to-action. Purple accent button.
- Body and headline font: 'Inter', a sans-serif font, for a clean and modern reading experience.
- Use modern and simple icons for navigation and contact information.
- Use a centered layout for the hero section, inspired by the 'Brooklyn Gilbert' design, with the profile image on the right and text on the left.
- Subtle animations using Framer Motion for transitions, hovers, and scrolling to enhance user engagement.
